# Pages

This directory contains page-level components that represent different routes/views in the application.

## Structure

- Each page component represents a specific route
- Pages compose multiple smaller components from the `components` directory
- Handle page-level state and data fetching

## Example

```
pages/
├── Home/
│   ├── Home.tsx
│   └── Home.test.tsx
├── About/
│   └── About.tsx
└── Profile/
    └── Profile.tsx
```
